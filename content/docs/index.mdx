Beta é um template full-stack TypeScript em monorepo projetado para você começar a construir funcionalidades desde o primeiro minuto — sem perder tempo decidindo arquitetura, configurando autenticação ou conectando camadas.

Toda a estrutura segue uma **arquitetura limpa orientada a domínios de negócio**: a lógica de negócio vive isolada no `core`, a infraestrutura implementa os contratos, e a API expõe tudo de forma type-safe do banco até o frontend. Autenticação, banco de dados, validação e estilização já vêm configurados e integrados.

## O que vem pronto

- **Monorepo com Turbo** — builds incrementais, cache inteligente e tasks paralelas
- **API type-safe de ponta a ponta** — oRPC conecta server e client com tipagem completa, sem code generation
- **Autenticação completa** — login, registro, sessões e proteção de rotas via Better Auth
- **Banco de dados com migrations** — Drizzle ORM com PostgreSQL, schema em TypeScript e RLS habilitado
- **Frontend moderno** — React 19 com TanStack Router (file-based) e TanStack Query para data fetching
- **Estilização com componentes** — Tailwind CSS + DaisyUI com suporte a temas
- **Deploy simplificado** — setup com Railway em um comando, ou traga seu próprio PostgreSQL
